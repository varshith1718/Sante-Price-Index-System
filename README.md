# Sante Price Index System

An Android application developed using Kotlin and Jetpack Compose for monitoring and managing market prices.

## Features

* User Login and Registration
* Vendor Profile Management
* Market Price Tracking
* Dashboard Interface
* Dark and Light Mode
* Firebase Integration
* Real-Time Data Storage

## Technologies Used

* Kotlin
* Android Studio
* Jetpack Compose
* Firebase
* Git & GitHub

## Project Structure

* `app/` - Main Android application source code
* `gradle/` - Gradle configuration files
* `build.gradle.kts` - Project build configuration
* `settings.gradle.kts` - Project settings

## How to Run

1. Clone the repository
2. Open project in Android Studio
3. Sync Gradle files
4. Run the application on emulator or Android device
